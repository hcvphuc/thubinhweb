## Bước 3: Đọc SKILL.md của skill được chọn

Sau khi xác định skill phù hợp, đọc file hướng dẫn chi tiết:

```
view_file: /Users/tuuyenlemai/Working/Antigravity/Thubinh_web/antigravity-awesome-skills/skills/<tên-skill>/SKILL.md
```

> **Lưu ý:** Nếu file `SKILL.md` không tồn tại, thử tìm file `.md` khác trong thư mục skill đó.

## Bước 4: Áp dụng skill vào công việc

Sau khi đọc xong skill, áp dụng kiến thức và hướng dẫn từ skill vào task hiện tại. Có thể kết hợp nhiều skills cho một task phức tạp.

---

## 💡 Mẹo sử dụng

- **Kết hợp skills**: Một task lớn có thể cần kết hợp nhiều skills. Ví dụ: `brainstorming` + `architecture` + `react-best-practices`
- **Ưu tiên chất lượng**: Luôn dùng `code-reviewer` hoặc `find-bugs` sau khi code xong
- **Cho project thubinh-web**: Ưu tiên `gemini-api-dev`, `react-best-practices`, `supabase-automation`
- **Tìm thêm skills**: Dùng `/skills <keyword>` để tìm skill không có trong danh sách trên
